Our Plain Page

Here, we have an example of an HTML file that is pretty plain.

With a few simple steps, we can add some style to our page.

  1. Step 1: Add the Bootstrap v3.3.7 stylesheet to the head of our HTML file.
    • You will notice that the font changes immediately
    • This will also give us access to pre-styled Bootstrap classes, like 'jumbotron' and 'container', which we will see in the following steps
  2. Step 2: Wrap the HTML of our body in a div with a class of 'jumbotron'
    • This automatically adds styling to the content, making it larger and bolder.
    • It also has a pre-defined background-color of '#eee'
  3. Step 3: Wrap the HTML of our body in a div with a class of 'container'
    • This automatically adds a margin to our page
  4. Step 4: Add a Navigation Bar to our body above our 'container' div
    • Some examples can be found here
  5. Step 5: Style our background using two custom CSS selectors.
